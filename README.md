🔋 Battery Stats Module

🇷🇺 Описание (RU)
Этот модуль предназначен для тех, кто хочет знать о своем аккумуляторе всё. В отличии от приложений по типу battery guru или ABattery модуль берет данные с ядра, а не из dumpsys.

🎉 Что нового в V4:

Полная емкость: Теперь отображается текущая максимальная емкость и заводская емкость в mAh (например, 4416/4610 mAh).

Обновленный UI: Проценты емкости теперь отображаются с небольшим отступом для лучшей читаемости.

Повышенная стабильность: Полный переход с sed -i на echo. Теперь обновление описания работает стабильнее на всех версиях HyperOS и кастомных прошивках.

Исправление багов: Решена проблема, когда описание модуля не обновлялось или не отображалось.



🇺🇸 Description (EN)
This module is designed for those who want to know everything about their battery. Unlike applications like battery guru or ABattery, the module takes data from the kernel, not dumpsys 

🎉 What's new in V4:
Real Capacity: Now displays current maximum capacity and factory capacity in **mAh** (e.g., 4416/4610 mAh).
 
Updated UI: Capacity percentages are now displayed with a slight offset for better readability.
 
Improved Stability: Completely switched from sed -i to echo. Module description updates are now more stable across all HyperOS versions and custom ROMs.
 
Bug Fixes: Fixed an issue where the description was not updated or simply did not appear.
 
🛠 Installation / Установка
1. Скачайте файл в релизах/Download the file from Releases.
2. Установите в своем рут менеджере/ Install via your root manager.
3. Перезагрузите свое устройство/Reboot your device.
4. Проверьте описание модуля в приложении/Check the module description in the app.
